# eggyengine

A project for a game engine built entirely in Zig in the same way Unreal Engine is built in C++. 

## Current projects:
| Library | Description |
| --- | --- |
| [vitellus](https://github.com/eggyengine/vitellus) | A render hardware interface in a Zig way |
| [eggenvector](https://github.com/eggyengine/eggenvector) | A SIMD-powered math library for advanced algebra |
| [candler](https://github.com/eggyengine/candler) | Common windowing interface to connect `vitellus` to windowing libraries |
| [clutch](https://github.com/eggyengine/clutch) | An ECS system inspired by bevy_ecs |
| [humpty](https://github.com/eggyengine/humpty) | Physics engine written entirely in Zig |
| [zeui](https://github.com/eggyengine/zeui) | A UI interface built with vitellus |
